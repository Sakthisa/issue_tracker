Sensor DX
DOCUMENTATION

SensorDX is an open-source standard-compliant software system that implements novel machine learning methods for automated quality control (QC) of hydro-meteorological sensor network data.

The goal of this project is to achieve a much higher degree of automated QC to permit a smaller number of experts to manage a much larger number of sensors.

Throughout this documentation, we have highlighted our powerful and robust API behind the SensorDX Dashboard (SDD), the SensorDX Issue Tracker (SDI), and SensorDX Sensor Management System (SMS).

Also, we went out of our way to give you an overview of the powerful and ultimate SensorDX algorithm that makes automated quality control possible. Who would have thought managing millions of sensors could be this fun? You're welcome :-)
